crunchy: Shiny Apps on Crunch

To facilitate building custom dashboards on the Crunch data platform <http://crunch.io/>, the 'crunchy' package provides tools for working with 'shiny'. These tools include utilities to manage authentication and authorization automatically and custom stylesheets to help match the look and feel of the Crunch web application.
