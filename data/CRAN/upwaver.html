upwaver: Access 'Upwave' API

'Upwave' <https://www.upwave.io/> is a task management app organising tasks on boards and cards. 'upwaver' is a wrapper around the 'Upwave' API <https://upwavehq.github.io/api/> that allows listing and updating those boards and cards.
