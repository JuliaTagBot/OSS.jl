knockoff: The Knockoff Filter for Controlled Variable Selection

The knockoff filter is a general procedure for controlling the false discovery rate (FDR) when performing variable selection. For more information, see the website below and the accompanying paper: Candes et al., "Panning for Gold: Model-X Knockoffs for High-dimensional Controlled Variable Selection", 2016, <arXiv:1610.02351>.
