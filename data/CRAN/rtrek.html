rtrek: Datasets and Functions Relating to Star Trek

Provides datasets related to the Star Trek fictional universe and functions for working with the data. The package also provides access to real world datasets based on the televised series and other related licensed media productions. It interfaces with Wikipedia (<https://www.wikipedia.org/>), the Star Trek API (STAPI) (<http://stapi.co/>), Memory Alpha (<http://memory-alpha.wikia.com/wiki/Portal:Main>), and Memory Beta (<http://memory-beta.wikia.com/wiki/Main_Page>) to retrieve data, metadata and other information relating to Star Trek. It also contains local datasets covering a variety of topics such as Star Trek universe species data, geopolitical data, and summary datasets resulting from text mining analyses of Star Trek novels. The package also provides functions for working with data from other Star Trek-related R data packages containing larger datasets not stored in 'rtrek'.
