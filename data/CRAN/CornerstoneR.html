CornerstoneR: Collection for 'CornerstoneR' Interface

Collection of scripts for interface between 'Cornerstone' and 'R'. 'Cornerstone' (<https://www.camline.com/>) as a software for engineering analytics supports an interface to 'R'. The scripts are designed to support an easy usage of this interface.
