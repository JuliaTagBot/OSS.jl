rrd: Import Data from a RRD (Round Robin Database) File

Makes it easy to import the data from a 'RRD' database (<https://oss.oetiker.ch/rrdtool/>) directly into R data structures. The resulting objects are 'tibble' objects or a list of 'tibble' objects, making it easy to manipulate the data.
