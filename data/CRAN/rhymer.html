rhymer: Wrapper for the 'Datamuse' API to Find Rhyming and Associated Words

Wrapper for 'Datamuse' API to find rhyming and other associated words. This includes words of similar meaning, spelling, or other related words. Learn more about the 'Datamuse' API here <http://www.datamuse.com/api/>.
