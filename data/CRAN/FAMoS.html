FAMoS: A Flexible Algorithm for Model Selection (FAMoS)

Given a set of parameters describing model dynamics and a corresponding cost function, FAMoS attempts to find the most suitable parameter subset in terms of a specified information criterion. Works on any cost function that returns a negative log-likelihood.
