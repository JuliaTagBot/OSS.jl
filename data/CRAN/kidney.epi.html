kidney.epi: Kidney Functions: Clinical and Epidemiological

Contains kidney care oriented functions. Current version contains only function for calculation of Kidney Donor Risk Index and Kidney Donor Profile Index for kidney transplant donors by Rao et al. (2009) <doi:10.1097/TP.0b013e3181ac620b>. Citation: Bikbov B. R open source programming code for calculation of the Kidney Donor Profile Index and Kidney Donor Risk Index. Kidney Diseases (2018) <doi:10.1159/000492427>.
