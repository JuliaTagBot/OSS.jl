HellCor: The Hellinger Correlation

Empirical value of the Hellinger correlation, which is a new measure of dependence between two continuous random variables that satisfies a set of 7 axioms that any valid measure of dependence should satisfy. More details can be found in Geenens and Lafaye De Micheaux (2018) <arXiv:1810.10276>.
